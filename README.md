## Video Recorder

포토부스 느낌으로 제작한 간단한 비디오 레코더입니다.  
실시간 카메라 미리보기, 영상 녹화, 다양한 필터 적용, 좌우반전 기능을 하나의 화면에서 직관적으로 사용할 수 있도록 구성했습니다.

사용자는 키보드와 버튼 조작만으로 **미리보기 모드와 녹화 모드 전환**, **필터 변경**, **거울 모드 설정**을 손쉽게 수행할 수 있습니다.  
또한, 현재 선택한 필터와 좌우반전 설정은 **실시간 미리보기 화면뿐 아니라 실제 저장되는 녹화 영상에도 동일하게 반영**됩니다.

## 주요 기능

- 실시간 카메라 화면 미리보기
- 영상 녹화 및 파일 저장
- 여러 종류의 포토부스 스타일 필터 제공
- 좌우반전(Mirror) ON / OFF 지원
- 버튼 기반의 간단한 UI 조작
- 미리보기 화면과 녹화 결과물에 동일한 효과 적용

## 조작 방법

| 입력 | 설명 |
|------|------|
| **Space** | Preview 모드와 Record 모드 전환 |
| **ESC** | 프로그램 종료 |
| **Filter 버튼 클릭** | 필터 선택 메뉴 열기 |
| **Mirror 버튼 클릭** | 좌우반전 ON / OFF 전환 |

## 동작 방식

### 1. Preview 모드
기본적으로 카메라 화면을 실시간으로 보여주는 모드입니다.  
이 모드에서는 사용자가 현재 화면 구도, 필터 적용 상태, 좌우반전 여부 등을 확인할 수 있습니다.

- 카메라 입력이 실시간으로 화면에 표시됩니다.
- 선택한 필터가 즉시 반영됩니다.
- Mirror가 켜져 있으면 거울처럼 좌우가 반전된 상태로 표시됩니다.
- 이 상태에서는 아직 영상 파일이 저장되지 않습니다.


### 2. Record 모드
녹화를 시작하는 모드입니다.  
`Space` 키를 눌러 Preview 모드에서 Record 모드로 전환하면, 현재 화면 상태 그대로 영상 녹화가 시작됩니다.

- 하단 카메라 버튼에 녹화 아이콘이 활성화됩니다.
- 실시간으로 들어오는 프레임이 파일로 저장됩니다.
- 현재 적용 중인 필터와 Mirror 설정도 함께 저장됩니다.
- 다시 `Space` 키를 누르면 녹화가 종료되고 Preview 모드로 돌아갑니다.


## 저장 방식

녹화된 영상은 프로그램 실행 중 사용자가 녹화를 시작한 시각과 현재 필터 이름을 기준으로 파일명이 생성되어 저장됩니다.

- 파일 이름 형식: `rec_YYYYMMDD_HHMMSS_[필터이름].mp4`

시간과 필터 이름이 함께 들어가서, 언제 어떤 필터로 찍은 영상인지 파일 이름만 보고도 알 수 있습니다.


## 필터

하단 왼쪽의 **Filter** 버튼을 누르면, **카메라 전체 화면이 작게 들어간** 필터 미리보기 카드들이 표시됩니다.  
각 카드에는 해당 필터가 적용된 전체 화면이 썸네일로 보이므로, 비교해 보며 원하는 카드를 클릭하면 해당 필터가 즉시 적용됩니다.

녹화 영상에도 동일하게 반영되는 실제 출력 효과입니다.

### 제공 필터 목록

- **Original**  
  보정이 들어가지 않은 원본 화면입니다.  
  카메라 입력 그대로의 자연스러운 영상을 확인할 수 있습니다.

- **Soft**  
  전체적으로 밝기를 높이고 부드러운 느낌을 주는 필터입니다.  
  화면이 과하게 선명하지 않고 은은하게 보이도록 만들어 포토부스 느낌을 강화합니다.

- **Warm**  
  붉은색과 노란색 계열을 살려 따뜻한 분위기를 주는 필터입니다.  
  인물 촬영 시 보다 편안하고 포근한 인상을 줄 수 있습니다.

- **Cool**  
  푸른색 계열을 강조하여 차분하고 시원한 톤을 만드는 필터입니다.  
  밝고 깨끗한 느낌보다는 조금 더 정돈되고 냉한 분위기를 연출할 때 적합합니다.

- **Glow**  
  밝은 부분이 은은하게 번지는 효과를 주는 필터입니다.  
  화면에 약한 빛 번짐 효과를 더해 몽글몽글하고 감성적인 분위기를 만듭니다.

- **Film**  
  대비를 약간 높이고 미세한 필름 그레인 느낌을 추가한 필터입니다.  
  디지털 화면보다는 아날로그 카메라 같은 분위기를 표현하는 데 적합합니다.

- **Pastel**  
  대비와 채도를 줄이고 밝기를 높여 부드러운 파스텔톤 느낌을 주는 필터입니다.  
  자극적인 색감보다는 연하고 화사한 분위기를 원하는 경우에 잘 어울립니다.


## 필터 적용 범위

선택한 필터는 다음 두 영역에 모두 적용됩니다.

- 실시간 미리보기 화면
- 실제 저장되는 녹화 영상

즉, 사용자가 화면에서 보고 있는 결과와 저장되는 결과가 최대한 일치하도록 구성되어 있습니다.


## 좌우반전 (Mirror)

하단의 **Mirror** 버튼을 누르면 좌우반전 기능을 켜거나 끌 수 있습니다.  
이 기능은 필터와는 별개로 동작하며, 사용자가 원하는 화면 방향에 맞게 독립적으로 설정할 수 있습니다.

### Mirror ON
거울을 보는 것처럼 화면이 좌우 반전되어 표시됩니다.
이 모드는 셀프 촬영이나 포토부스 스타일 미리보기에 익숙한 사용자에게 자연스럽게 느껴질 수 있습니다.

### Mirror OFF
실제 카메라 방향 기준으로 화면이 표시됩니다.
이 모드는 녹화 결과물을 일반적인 카메라 영상처럼 자연스럽게 남기고 싶을 때 적합합니다.


## Mirror 기능의 특징

- 필터와 독립적으로 ON / OFF 할 수 있습니다.
- Preview 모드와 Record 모드 모두에 동일하게 적용됩니다.
- Mirror가 켜진 상태에서 녹화를 시작하면, 저장되는 영상도 같은 방향으로 기록됩니다.


## 사용 예시

1. 프로그램을 실행하면 카메라 미리보기가 나타납니다.
2. `Filter` 버튼을 눌러 원하는 필터를 선택합니다.
3. 필요하면 `Mirror` 버튼으로 좌우반전 여부를 조정합니다.
4. 화면 구도를 확인한 뒤 `Space` 키를 눌러 녹화를 시작합니다.
5. 녹화 중에는 하단 카메라 버튼의 상태를 통해 녹화 여부를 확인할 수 있습니다.
6. 다시 `Space` 키를 누르면 녹화가 종료되고 파일이 저장됩니다.
7. `ESC` 키를 누르면 프로그램이 종료됩니다.
